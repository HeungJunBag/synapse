# 공통 도메인 원칙 (DOMAIN-COMMON-CONSTITUTION)

## 1. 개요

시냅스의 모든 도메인에 공통으로 적용되는 원칙을 정의한다.

## 2. 핵심 원칙

- **데이터 소유권**: 모든 사용자 데이터는 생성한 사용자만 접근할 수 있다. RLS로 강제한다.
- **서버 우선**: 데이터 접근은 서버(Server Actions, Server Components)를 통한다. 클라이언트에서 직접 DB를 조작하지 않는다.
- **타입 안전성**: 모든 DB 데이터는 TypeScript 타입으로 정의된다. `any` 타입 사용을 최소화한다.
- **에러 전파**: DB 에러는 조용히 삼키지 않고 throw하여 호출자가 처리하게 한다.
